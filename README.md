# 📝 Todo App

A dynamic, single-page task management application built using **JavaScript modules and Webpack**.  
The application handles complex DOM manipulation, object-oriented data structures, and persistent state without relying on a frontend framework.

The project focuses on **factory functions, state management, local storage, and modern frontend tooling**.

## Live Demo

[View on GitHub Pages](https://yourusername.github.io/todo-app/)

## Features

- **Single Page Application structure** using JavaScript ES6 modules.
- **Dynamic DOM Rendering** to handle real-time UI updates for tasks and projects.
- **Local Storage Integration** to persist user data and projects across browser sessions.
- **Two-Pane Layout** utilizing CSS Flexbox for a clean, professional sidebar navigation system.
- **State Management** allowing users to seamlessly switch between a master "Home" view and isolated project folders.
- **Full CRUD Functionality** for tasks, including inline editing for titles, descriptions, due dates, and priorities.
- **Date Formatting** handled securely via the `date-fns` npm library.
- **Modular Architecture** with strict separation of logic (App Controller, Screen Controller, Data Factories).

## Built With

- **HTML5**
- **CSS3**
  - Flexbox
  - CSS Custom Properties (Theming)
- **JavaScript (ES6 Modules)**
  - Factory Functions
  - DOM Manipulation
  - Event Handling
- **Webpack**
  - Asset bundling
  - Dependency management
- **npm Libraries**
  - `date-fns` (Date parsing and formatting)
